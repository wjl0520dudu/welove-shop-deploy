## 示例：护肤知识片段

```markdown
# 肤质分类与护理指南

## 五大肤质类型

### 1. 干性皮肤
- 特征：毛孔细小，皮肤紧绷，容易脱皮、产生细纹
- 护理重点：深层保湿、滋润修复
- 推荐成分：透明质酸、神经酰胺、角鲨烷、甘油
- 避免：含酒精的控油产品、皂基洁面

### 2. 油性皮肤
- 特征：毛孔粗大，T区出油明显，容易长痘、黑头
- 护理重点：控油保湿、清洁毛孔
- 推荐成分：水杨酸、烟酰胺、茶树精油、高岭土
- 避免：过于滋润的面霜、厚重的防晒

## 护肤步骤（早晚通用基础版）
1. 洁面 → 2. 爽肤水 → 3. 精华 → 4. 眼霜 → 5. 乳液/面霜 → 6. 防晒（白天）
```

## 编写规范

1. **一级标题（#）**：文档总标题
2. **二级标题（##）**：分类/章节
3. **三级标题（###）**：具体知识点条目
4. **列表（-）**：列举特征、成分、优缺点
5. **表格**：对比不同选项
6. **引用（>）**：重要提示或说明
